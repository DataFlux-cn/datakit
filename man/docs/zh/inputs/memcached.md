Memcached 采集器可以从 Memcached 实例中采集实例运行状态指标，并将指标采集到观测云，帮助监控分析 Memcached 各种异常情况。

## 前置条件 {#requirements}

- Memcached 版本 >= `1.5.0`。已测试的版本：
    - [x] 1.5.x
    - [x] 1.6.x
