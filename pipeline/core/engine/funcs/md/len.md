### `len()` {#fn-len}

函数原型：`len(val=required)`

函数说明：计算 string 字节数，map 和 list 的元素个数。

参数:

- `val`: 可以是 map、list 或 string

示例:

```python
len([1, 2, "3"])

len({"a": 1, "b": 2, "c": []})

c = {"a": 1, "b": 2, "c": "123"}
len(c)
len(c["c"])
